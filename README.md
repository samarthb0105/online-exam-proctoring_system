\# 🎓 Online Exam Proctoring System



\## 📌 Description



This project is an AI-based online exam proctoring system that monitors students using computer vision techniques.



\## 🛠 Technologies Used



\* Python

\* OpenCV

\* MediaPipe

\* Machine Learning



\## 🚀 Features



\* Face detection

\* Eye blink detection

\* Gaze tracking

\* Suspicious activity monitoring



\## ▶️ How to Run



1\. Create virtual environment:

&#x20;  python -m venv venv



2\. Activate:

&#x20;  venv\\Scripts\\activate



3\. Install dependencies:

&#x20;  pip install -r requirements.txt



4\. Run:

&#x20;  python gui\_proctor.py



\## 📂 Project Structure



\* gui\_proctor.py → Main application

\* blink\_detector.py → Blink detection

\* gaze\_detector.py → Eye tracking

\* face\_detector.py → Face detection
